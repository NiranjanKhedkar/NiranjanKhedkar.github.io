05 — Projects

Things I've built

Project Hail Mary — Offline Translator

Personal

Inspired by Andy Weir's novel — record any sound, label it with an English word, then play that sound back and have it translated. Runs 100% offline. Extracts 40 MFCC features per audio clip via Librosa, stores them in a FAISS vector index, and retrieves the closest match using L2 distance. No cloud APIs, no network calls.

PythonStreamlitPyAudio LibrosaFAISSONNX Runtime
